
# Shrendar

This will be a metal archives clone but focused on birthday/other anniversaries, and tolerant of rock and punk.

## Tech Stack

**Backend**: Spring Boot, kotlin  
**Frontend/Web** - react.js  
**Mobile - android** - jetpack compose  
**Mobile - ios**  - swift


## Roadmap

- Login/registration with password, e-mail, username changes and logs - done

- Wiki pages

- User interactions with Wiki

- User interaction with other Users

- Frontend

- Mobile apps
